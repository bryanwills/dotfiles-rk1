## ⚠️ Notice to anyone cloning this repository:**

### Usage

This repository mirrors a personal, daily-driver Arch Linux setup and live development environment. There is no automated installation script. Feel free to browse, cherry-pick individual configurations, or manually copy specific scripts and widgets to suit your own workflow. 

I am actively tweaking, breaking, and rewriting these configurations all the time. 

If you clone or pull updates from this repo, please be aware that structural changes might break your local setup. 

**Use and update at your own risk!**

> *NOTE: If there is enough interest and requsts in my dotfiles, I might think about restructuring the repo and make it installable with a script.* 

**Leave a message in the Discussions on GitHub should you have anything you want to ask or report and issue.**
